# 📍 Flutter CEP

> Um aplicativo mobile desenvolvido em Flutter para consulta rápida de endereços a partir do CEP.

---

## 🚀 Sobre o Projeto

O **Flutter CEP** é uma aplicação prática que permite ao usuário digitar um CEP brasileiro e obter instantaneamente as informações de endereço correspondentes (como Logradouro, Bairro, Localidade e UF). Ideal para automação de formulários ou consultas rápidas de localização.

*(Dica: Se você utilizou alguma API específica como ViaCEP, AwesomeAPI ou BrasilAPI, ou alguma biblioteca de gerenciamento de estado como Bloc, Provider ou MobX, adicione aqui!).*

## 🛠️ Tecnologias e Pacotes Utilizados

* **Framework:** [Flutter](https://flutter.dev/) (Dart)
* **Requisições HTTP:** `http` ou `dio` *(Ajuste conforme o que usou)*
* **Gerenciamento de Estado:** [ex: setState / Provider / Bloc]

## ✨ Funcionalidades Propostas

- [ ] Consulta em tempo real de CEPs válidos.
- [ ] Validação visual de campos (evitar caracteres especiais ou tamanho inválido).
- [ ] Exibição organizada dos dados (Rua, Bairro, Cidade, Estado).
- [ ] Tratamento de erros para CEPs inexistentes ou falhas de conexão.
